RaySon focuses on practical CLI automation and maintainable release workflows.

## CLI capabilities

- Commandized script execution through one entrypoint (`rayson`)
- Finance export flows, including Smartsheet-oriented output formatting
- Environment and infrastructure helpers (`wsl startup`, `mongo ssl-setup`)
- Script root auto-detection with manual override support

## Maintainer workflow

- Source-controlled version values with sync tooling
- Release-branch snapshot builds for validation
- Tag-based publish workflow for official GitHub Releases
- GitHub Pages documentation deployment from Actions

## Project direction

- Windows-first execution for existing script packs
- Incremental migration from shell scripts to native Go commands
- Contributor-friendly docs and release runbooks
